Capstone Project - Wearable Radar
4-person team, project sponsored by Bose

The goal of the project is to create a fully portable, wearable device that will warn the wearer of impending collisions. The radar board that is being used currently is the IWR1642BOOST by Texas Instruments. Currently, an object tracking function and a basic collision detection algorithm have been developed

Expected Completion - April 2018

Chess Game
Solo project, created in Java

A GUI-based two-player chess game. Logic was implemented so that players could not make illegal moves. The game is also able to determine when a player is in check or checkmate

TRON Game
2-person team, created in hardware using VHDL, implemented on FPGA

Implementation of the classic TRON videogame in VHDL. Design utilizes a finite-state machine to address and write pixel colors to VRAM, accept user input via touch-screen LCD display, move player and AI characters, and determine whether or not the player or AI has lost the round using conditional logic. The enemy AI was implemented in multiple difficulties, with enemy movement becoming more sophisticated as difficulty increases. Character sprites created and drawn using ROM connected to the main design

Robot-Box Simulation
2-person team, created in C and designed for Ubuntu OS

Simulates randomly-generated robots pushing boxes into assigned doors. The simulation is multi-threaded and utilizes semaphores to control access to common resources

Total College Planner App
2-person team, app developed for iPhone

Created a Total College Planner app for iPhone which allowed the user to create, modify and view a class schedule as well as a financial budget. The app also had additional note-taking functionality, where the user could create small text documents that could be viewed inside the app
